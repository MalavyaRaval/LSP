## Steps to Run this project  
git clone https://github.com/MalavyaRaval/LSP

### Backend Setup  
cd backend  
npm install dotenv  
npm install multer  
npm audit fix  

### Run Backedn  
cd backend  
npm start  

### FrontEnd Setup  
cd frontend  
npm install  
npm audit fix  

### Start Frontend  
cd frontend  
npm run dev
